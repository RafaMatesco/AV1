# Aeronave Management System (AV1)

Este é um sistema de gerenciamento de aeronaves (Aerocode) desenvolvido em TypeScript.

## Como executar o projeto

Certifique-se de ter o **Node.js** instalado na sua máquina.

### 1. Instalar dependências

Instalar dependecias necessárias

```bash
npm install
```

### 2. Compilar o projeto (Opcional)

Caso queira gerar a versão compilada em JavaScript na pasta `dist`:

```bash
npm run build
```

### 3. Executar o projeto

```bash
# Iniciar a aplicação
npm start
```
